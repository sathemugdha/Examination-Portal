
#  Online Examination System (PHP & MySQL)


The **Online Examination System** is a web-based platform designed to automate traditional exam processes.  
It allows students to take objective-type exams online while enabling administrators to manage questions, monitor performance, and generate results efficiently.  

This project focuses on improving **speed**, **accuracy**, and **eco-friendliness** by reducing paper usage and automating the entire evaluation process.

---

##  Project Objectives
- Create a **secure and efficient** platform for conducting online exams.  
- Automate **answer checking** and **result generation**.  
- Allow administrators to **add, edit, and delete** questions and topics.  
- Provide an intuitive interface for both students and admins.  
- Reduce human effort and save time in exam management.  

---

##  Features

###  Student Module
- Secure login using **user ID** and **password**.  
- Attempt exams within a **specified time duration**.  
- Modify answers before submission (within time limit).  
- Automatic submission when the time ends.  
- Instant result calculation and feedback.  

###  Admin Module
- **Admin Dashboard** for managing the system.  
- Add, update, or delete **topics**, **questions**, and **user profiles**.  
- View exam results and analyze student performance.
  
###  Additional Highlights
- Real-time **exam timer**.  
- Fully automated evaluation and validation.  
- Error-free and consistent grading.  
- Environment-friendly digital platform.  

---

##  Technology Stack

### Frontend:
- **HTML** - Structure and layout  
- **CSS** - Styling and responsiveness  
- **JavaScript** - Dynamic features (timers, validation)

### Backend:
- **PHP** - Server-side scripting and logic handling  
- **MySQL** - Database management for users, questions, and results

---

##  Software Requirements
You can use any of the following local web servers:

- **WAMP** (Windows)
- **XAMPP** (Cross-platform)
- **MAMP** (macOS)
- **LAMP** (Linux)

Each package includes **Apache**, **PHP**, and **MySQL** for running PHP projects locally.
